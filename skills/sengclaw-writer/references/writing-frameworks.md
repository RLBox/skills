# 写作框架详解

## 1. 口语化自然流（默认推荐）

灵感来源：罗振宇「口语化表达」演讲精华

**核心原则**：
- 先天语言 vs 后天语言：用自然说话的节奏写，读者读起来不费力
- 对象化语言：脑子里始终有一个具体的人在读，像给好朋友发消息
- 线性交付：有起点有终点，读者跟着走，每句话都有下文牵引

**结构模板**：
```
[开场]：一个具体的场景/时刻/问题（2-3句）
[转折]：原来我以为...但发现...（制造期待）
[展开]：用3-4个小故事或场景具体说明
[收尾]：一句话总结感悟 + 问读者一个问题
```

**语言技巧**：
- 用"你"来拉近距离："你有没有发现..."
- 用广东人说话方式：直接、务实、不绕弯
- 多用数字和具体细节：不说"很贵"，说"1999块"
- 类比生活常见事物：把复杂概念比喻成大家都懂的东西

## 2. OmniThink 干货流

适合：知识科普、经验总结、方法论

**执行步骤**：
1. **信息扩展**：先brainstorm所有相关知识点（内部）
2. **反思整合**：找出最稀缺、最有洞见的点，删低价值信息
3. **信息树构建**：Why（为什么重要）→ How（怎么做）→ Warning（坑）→ Metric（数据）
4. **大纲**：3-5个清晰标题
5. **撰写**：每段信息密度最大化

**输出特征**：
- 有结构感：用编号或标题分层
- 数据说话：有具体数字或案例
- 直接给结论：不绕弯子，先给答案再解释
- 信息量密：每段都有收获感

## 3. 情绪共鸣流（咈蒙风格升级版）

适合：生活感悟、痛点共鸣、价值观输出

**情绪递进模型**：
```
模式中断 → 植入共鸣 → 路径引导 → 情绪闭环
```

**开头技巧**：
- 泛化共鸣："所有...的人，都...""99%的人不知道..."
- 读心术："你是不是也有这种感觉..."
- 意外反转："我以为...结果..."

**必须包含**：
- 扎心一句：让人停下来想想自己
- 重新定义：把一个普通认知重新解释
- 情绪出口：愤怒→共情→希望→行动冲动
- 结尾金句：一句话，让人想转发

## 4. 故事叙述流

适合：产品测评、生活记录、真实经历

**基本结构**：
```
[时间/地点/人物]：给故事设置舞台
[起因]：为什么会有这次经历
[过程]：具体发生了什么（有细节有对话）
[转折]：最让人意外的一个瞬间
[感悟]：从这个故事里读者能得到什么
```

**细节法则**：
- 用感官细节：看到了什么、听到了什么、感受到什么
- 有对话：如果有真实对话，原话引用
- 时间戳：具体的时间让故事更可信
- 配角存在：周周、小龙虾、家人、朋友都可以是故事元素

---

## 5. 第一性原理流

适合：深度思考、认知刷新、颠覆性观点

**执行模板**：
```
[现象]：大家通常怎么看这件事？
[质疑]：但这个"常识"真的对吗？让我重新想一想...
[拆解]：回到最基本的事实是...
[重建]：从这些基本事实出发，我得出...
[结论]：所以我认为...（反常识的新观点）
[行动]：基于这个认知，我打算...
```

**张润胜专属角度**：
- 用AR眼镜的视角看世界："当我带着乐奇眼镜看这件事时..."
- 广东人的实用主义："这东西到底能不能赚钱？"
- 早期探索者的优势："我比大多数人早用了X个月..."

# 平台排版规范

## 公众号规范

### 标题规范
- 字数：15-25字最佳（太短没吸引力，太长被截断）
- 类型选一：
  - 悬念型："我用了X天，发现一个让所有人都震惊的事"
  - 干货型："5个让你马上能用的XXX技巧"
  - 故事型："那天我把乐奇眼镜戴去菜市场，发生了..."
  - 观点型："为什么大多数人对AR眼镜的理解是错的"
  - 情感型："用了一年AR眼镜，我后悔没早点..."

### 正文规范
- **总字数**：800-1500字（太短显得不用心，太长掉读）
- **段落长度**：每段不超过150字，3-5行
- **分隔方式**：每个段落之间空一行
- **小标题**：建议每400字设一个小标题（加粗）
- **配图数量**：3-5张，首图最重要

### 结尾格式
```
---
你有没有类似的经历？在评论区告诉我 👇

（可选：关注我，每天分享一个科技新发现）
```

### SEO关键词建议（根据内容选）
AR眼镜、乐奇眼镜、AI工具、副业赚钱、科技评测、智能硬件

---

## 小红书规范

### 标题规范（超重要！）
- 字数：不超过30字
- 必须有emoji（1-3个）
- 制造好奇或共鸣：
  - "我用AR眼镜做了什么？被同事看傻了🥽"
  - "小红书没人讲的乐奇眼镜真实评测👀"
  - "月入xxx的副业，我是怎么发现的💰"
  - "带着AR眼镜遛狗是种什么体验？"

### 正文结构
```
[黄金开头 - 前3行]（展开前的可见内容，最关键！）
要么扎心，要么好奇，要么惊喜
让人忍不住点"展开"

[正文分点]
✅ 第一点：简短说清楚
✅ 第二点：加个细节或案例
✅ 第三点：有意外或转折
（根据内容3-6点，每点1-3行）

[结尾互动]
你试过/有没有/你觉得...？
评论区聊聊👇

[话题标签]
#标签1 #标签2 #标签3 ... （5-8个）
```

### 话题标签库（根据内容选）
**科技类**：#AR眼镜 #乐奇眼镜 #智能硬件 #科技好物 #AI工具 #数码测评 #科技爱好者 #智能穿戴

**生活类**：#广东生活 #养狗日记 #生活记录 #真实体验 #日常分享

**副业类**：#副业赚钱 #AI创业 #内容创作 #赚钱思路 #搞钱日记

**内容类**：#好物分享 #测评 #种草 #真实评测 #用后感

### 配图建议
- 小红书最佳尺寸：1:1 或 4:3
- 首图最重要：要么颜值高要么信息量大
- 多图时：第一张吸引点击，最后一张引导关注
- 建议图数：3-9张（9张最大化展示机会）

### 发布时间建议
- 最佳时段：早7-9点、午12-14点、晚20-22点
- 避开周一早上（大家没心情刷手机）
- 周末下午流量高峰

## 两平台差异对比

| 维度 | 公众号 | 小红书 |
|------|--------|--------|
| 字数 | 800-1500字 | 300-500字 |
| 风格 | 深度、完整 | 轻快、视觉化 |
| 标题 | 逻辑驱动 | 情绪/好奇驱动 |
| 配图 | 补充说明 | 核心内容 |
| 互动 | 评论、在看 | 点赞、收藏、评论 |
| 适合内容 | 深度分析、长文观点 | 种草、经验分享、好物推荐 |

# XHS Comment Diversion（评论导流方法论）

> **背景案例**：某用户曾在热门视频下发纯情绪共鸣评论，获得 3000+ 赞，但**零账号转化**。
> **核心教训**：互动 ≠ 转化。情绪共鸣型评论能拿赞，但不会让人想关注你。

## 一、底层逻辑：为什么要做评论导流？

### 1.1 评论区的流量价值
- 小红书用户刷完笔记后，**80% 会点开评论区找答案**
- 一条高赞评论的曝光量 **≈ 一篇普通笔记**
- 优质评论可**持续吸粉 1-3 个月**（算法持续推送高互动评论）
- 在爆款笔记评论区借力，**零成本触达精准用户**

### 1.2 转化漏斗模型
```
曝光（用户看到评论）
    ↓
好奇心触发（钩子生效）
    ↓
点击主页（身份锚点生效）
    ↓
留下关注（主页承接到位）
    ↓
私信/互动（转化）
```

### 1.3 三大核心公式

| 公式 | 说明 |
|------|------|
| **精准场景 × 钩子话术** | 在对的地方说对的话，不是乱撒网 |
| **身份锚点 + 悬念/反差** | 让人知道你是谁，还要让人想主动来找你 |
| **主页承接 + 私信闭环** | 导流进来之后要接得住，不然前功尽弃 |

## 二、评论选区策略（精准找对战场）

### 2.1 选区标准

| 维度 | 标准 |
|------|------|
| **垂直度** | 只选与用户目标赛道相关的爆款（见 EXTEND.md `comment_diversion.target_niches`） |
| **热度门槛** | 点赞 ≥ 1000，评论 ≥ 100 |
| **活跃窗口** | 优先发布 ≤ 7 天的笔记，评论区仍在活跃期 |
| **避开红线** | 避开明星/明显营销类笔记（举报风险高） |

### 2.2 通用黄金战场类型

- **垂直领域创业/副业类笔记**：受众自带「想改变现状」标签
- **打工人情绪共鸣类**：反差钩子最有效
- **技能/工具教程类**：悬念钩子高效
- **逆袭/转型类**：反差对比天然匹配

> 具体目标赛道由用户在 EXTEND.md `comment_diversion.target_niches` 配置。

### 2.3 黄金时间窗口

- **抢占时机**：爆款笔记发布后 **6 小时内**，进入前排位置
- **高活跃时段**：早 8-9 点 / 午 12-13 点 / 晚 20-22 点
- **风控节奏**：同一账号 1 小时内评论不超过 3 条，避免营销号判定

## 三、四大钩子类型（通用框架）

> 生成具体评论时，用 EXTEND.md 中的用户信息填充 `{placeholder}`。

### Type A：身份植入型（日常主力）

> **目标**：让人知道你是谁，点进主页了解你
> **适用**：中性/共鸣类笔记，与用户赛道相关

**公式**：`[{audience}身份标签] + [情绪共鸣/认同] + [价值暗示，引导主页]`

**通用结构**：
```
[说明你是做什么的/在做什么阶段]
[对这条笔记的情绪认同，1句]
[价值暗示，如「主页有记录整个过程」]
{signature}
```

**生成示例**（填充后）：
```
在搞{product}的路过，这条说到心坎了
[一句情绪共鸣]
（主页有记录折腾过程）{signature}
```

---

### Type B：悬念钩子型

> **目标**：引发好奇心，让用户主动来找你
> **适用**：干货/教程类笔记，受众有「我也想学」的意愿

**公式**：`[痛点共鸣] + [暗示我找到了解法] + [留悬念/引导主页]`

**通用结构**：
```
[与笔记话题的痛点共鸣，1句]
[转折：我后来做了什么/发现了什么，不说答案]
[引导：折腾记录在主页，感兴趣可以看看]
{signature}
```

---

### Type C：反差对比型

> **目标**：用强烈前后对比制造落差感，触发「我也能这样吗」
> **适用**：励志/逆袭/焦虑/打工人类笔记，情绪浓度高

**公式**：`[前状态，来自{background}] + [转折点] + [后状态暗示，不过分夸大]`

**通用结构**：
```
[前状态，简短有代入感，取自identity.background]
→ [发生了什么转变，1句]
不是鸡汤，真实发生的，主页有完整过程 👆
{signature}
```

---

### Type D：福利赠品型（备用）

> **目标**：用具体资源吸引点击
> **适用**：有明确资料包/清单时才用，不能凭空承诺

**公式**：`[具体福利名称] + [效果背书] + [主页自取]`

**通用结构**：
```
我整理了一份《[具体清单名称]》（主页自取）
[一句话说清楚是什么/解决什么问题]
亲自踩坑整理的，不是转载{signature}
```

> ⚠️ **禁用词**：「免费」「加微信」「私信我」→ 改用「主页自取」「主页有」

## 四、主页承接要求

评论做得再好，主页不到位 = 漏斗底部漏水。

### 4.1 主页三要素

| 要素 | 要求 |
|------|------|
| **昵称** | 垂直领域 + 身份标签（来自 EXTEND.md `account.nickname`） |
| **简介** | 一句话说清你能给用户什么价值（来自 `account.bio`） |
| **置顶笔记** | 强钩子内容（来自 `account.pinned_post`） |

## 五、私信闭环话术（通用版）

用户私信后，分步引导，不要直接发联系方式：

```
宝子你好～
看到你对 [XX话题] 感兴趣
我把相关内容整理到了主页，点进去就能看到完整过程 👆

如果想深入聊聊，可以继续发我
不同阶段适合的路径不一样，先聊聊你现在的情况？
```

## 六、三大雷区

| 雷区 | 说明 | 替代方案 |
|------|------|---------|
| **敏感词** | 禁用「免费」「加微信」「私信我」 | 用「主页有」「主页自取」 |
| **复制粘贴** | 相同话术重复触发平台去重 | 每条做细微变化 |
| **钩子与主页脱节** | 评论说送资料，主页却没有 | 先准备好主页内容再去评论 |
| **无身份锚点** | 纯情绪评论，拿赞但零转化 | 每条都要有身份标识 |
| **过度频繁** | 1小时内超过3条易被判营销号 | 控制节奏，每天目标见 EXTEND.md |

---

## 七、评论效果评估指标

| 指标 | 健康值 | 说明 |
|------|--------|------|
| 主页点击率 | 每 100 赞 → 5+ 主页访问 | 低于这个说明钩子无效 |
| 关注转化率 | 每 10 次主页访问 → 1+ 关注 | 低则说明主页承接有问题 |
| 私信转化 | 每周 3-5 条私信 | 核心变现入口 |

## 八、每日评论导流 SOP

```
1. 读取 EXTEND.md，确认身份配置
2. 搜索 EXTEND.md `comment_diversion.target_niches` 里的目标赛道
3. 筛选：点赞≥1000 + 发布≤7天 + 评论活跃
4. 根据笔记情绪基调选钩子类型（A/B/C/D）
5. 填充用户信息（{background}/{product}/{signature} 等）生成评论
6. 每条做细微变化，确保有身份锚点
7. 每日目标条数见 EXTEND.md `comment_diversion.daily_limit`
8. 记录当天评论内容和目标帖子链接
9. 次日检查：哪条获得最多主页点击？
10. 每周复盘：效果好的话术由用户自己沉淀到 EXTEND.md
```
